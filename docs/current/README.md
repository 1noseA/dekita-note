# Dekita Note (できたノート)

Minimal Viable Product for tracking daily small achievements.

## Features
- **Daily Records**: Log your achievements and health checks (Medication, Period, Hospital).
- **Calendar View**: Visual history of your productivity with color-coded days.
- **Data Persistence**: All data is saved locally in your browser.
- **Data Privacy**: No external servers, data stays on your device.

## Tech Stack
- React + TypeScript + Vite
- TailwindCSS
- React Router DOM
- LocalStorage for persistence

## Setup
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```

## Usage
- Open the app.
- Login as `User 1` or `User 2` (Simulated multi-user).
- Click on a date in the calendar or "Record Today" to add entries.
- Toggle health check visibility in Settings.
